You can configure Blacklight from here. 

Blacklight.configure(:environment) do |config| end

:shared (or leave it blank) is used by all environments. You can override a shared key by using that key in a particular environment’s configuration.

If you have no configuration beyond :shared for an environment, you do not need to call configure() for that envirnoment.

For specific environments: 

Blacklight.configure(:test) {}
Blacklight.configure(:development) {}
Blacklight.configure(:production) {}
